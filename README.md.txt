### Gestión de bases de datos (1º ASIR)

En esta segunda práctica aprenderemos a usar mongoDB y Visual Studio Code para
la creación y manejo de bases de datos. En este repositorio econtrará.

> 1. Un ***manual de instalación*** donde explicaremos paso por paso la instalación de MongoDB y Visual Studio Code. 
> 2. Una breve explicación teórica sobre los ***primeros comandos de Mongo*** y la ***creación de una base de datos***.
> 3. Una serie de ***documentos .js***: dos colecciones y una hoja de consultas con ejemplos de posibles búsquedas.
